## 📊 Overview

This project analyzes hospital patient data to identify key factors influencing readmissions and high-risk cases. The goal is to uncover patterns and provide actionable insights for improving patient outcomes.

---

## 🎯 Objectives

* Understand patient readmission patterns
* Identify high-risk patient groups
* Analyze key drivers such as hospital stay, medications, and emergency visits
* Build an interactive dashboard for decision-making

---

## 🧠 Key Insights

* High-risk patients show significantly higher readmission rates
* Longer hospital stays are associated with increased readmissions
* Higher medication usage indicates increased patient severity
* Emergency admissions contribute to higher readmission likelihood

---

## 📈 Dashboard Features
